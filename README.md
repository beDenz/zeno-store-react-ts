# Zeno store

### React / TypeScript/ SCSS

##### Firebase: https://zenostorereactts.web.app/

- Хостинг и БД на Firebase
- Адаптив на мобильные телефоны до 375px
- Реализован функционал магазина, с выводом списка товаров, в виде списка или сетки. Реализована бесшовная сортировка по категориям, фильтрация по цене. Добавление в корзину и вишлист. Внутри карточки можно изменить количество, или удалить товар.
  Выборка товара берется из БД на Firebase в виде массива объектов, в которых находятся детали товара.
